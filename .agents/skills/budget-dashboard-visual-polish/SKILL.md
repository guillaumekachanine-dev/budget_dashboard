Visual polish checklist:
- Reduce visual noise.
- Use a clear hero section per page.
- Use compact cards with strong hierarchy.
- Prefer 2-column stat grids on mobile where readable.
- Avoid large tables on mobile; use timeline/list cards.
- Use subtle gradients only for hero/primary cards.
- Use borders and shadows sparingly.
- Keep colors meaningful, not decorative.
- Use consistent rounded corners.
- Ensure readable contrast.
- Check small-screen behavior.
